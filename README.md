# Constraint Satisfaction Problems (CSP) in Prolog

This repository is a collection of various problems from [CSPLib](https://www.csplib.org/). My goal is to implement efficient solutions with Declarative Programming.


###  [N-Queens Problem](https://www.csplib.org/Problems/prob054/)
* **Description:** Place N queens on an NxN chessboard so that no two queens threaten each other.
* **Key Learnings:** Difference between *Generate & Test* and *Interleaving/Backtracking*.
* **Performance (N=12):** 25,800 inferences (Optimized) vs. 89,400,000 (Naive).
